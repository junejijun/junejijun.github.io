数据类型和运算符——Java（一）

1.标识符使用规范

1.1规则

  • 标识符必须以字母、下划线_和美元符号$开头；
  • 标识符的其他部分可以是字母、下划线_、美元符号$和数字的任意组合；

1.2使用规范

  1. 类名的标识符：每个单词的首字母大写，如GoodMan;
  2. 方法的标识符：第一个单词的首字母小写，后面单词的首字母大写，遵循驼峰原则，如eatFood;
  3. 变量的标识符：第一个单词的首字母小写，后面单词的首字母大写，遵循驼峰原则，如studentAge;
  4. 常量的标识符：全部大写，单词间用下划线隔开，如MAX_Value;

注：中文也属于字母的一种；关键字（保留字）不可作为标识符

2.变量

1.1基本类型与引用类型

  • 基本类型：分为三类——整数类型、字符类型和布尔类型；
  • 引用类型：又称为对象类型，分为五类——类、接口、数组、枚举和标注；

注：基本数据类型根据数据类型不同，其大小也不一样；引用数据类型大小统一，占4个字节，用于存放引用对象地址（1bit(字节)=8byte(位)）；

1.2变量分类

  • 局部变量：从属于方法/语句块
  • 成员变量（实例变量）：从属于对象
  • 静态变量（类变量）：从属于类

1.3基本数据类型

注：

  1. Java语言的整型常数默认类型为int型，声明long型常量需在其后加‘L’；
  2. 浮点数值默认类型为double型，声明float型常量需在其后加’F’；
  3. 字符型数值用单引号，字符串数值用双引号；（字符型数据占2个字节）；
  4. 布尔型数据占1位；

3.运算符

3.1算数运算符

3.1.1整数运算
  • 如果两个操作数中有一个为long型，则结果也为long型；
  • 如果两个操作数都不为long型，则结果为int型，即使操作数全为short型或byte型；
3.1.2浮点运算
  • 如果两个操作数中有一个为double型，则结果也为double型；
  • 如果两个操作数都为float型，则结果为float型；

3.2赋值运算符

a *= b+3 等价于a = a*(b+3)

3.3逻辑运算符

  • 逻辑与“&”和逻辑或“|”；
  • 短路与“&&”和短路或“||”；

3.4位运算

  • “<<”左移运算符，左移一位相当于乘2：3<<2等价于3*2*2；
  • “>>”右移运算符，右移一位相当于除2：12>>2等价于12/2/2;

3.5字符串连接符

“+”运算符两侧的操作中只要有一个是字符串，系统会将另一个操作数转换为字符串，然后进行连接；

3.6条件运算符

形如：x ? y : z
