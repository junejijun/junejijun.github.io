面向对象基础篇——Java（三）

1.对象与类

1.1对象

  • 对象是一种数据结构，本质上就是一种数据的组织方式；
  • 在内存上，对象就是一个内存块，存放相关的数据集合；

1.2类

1.2.1定义

类是用于描述同一类型的对象的一个抽象概念，类中定义了这一类对象所对应的共同属性和方法；

1.2.2 组成

对于一个类来说，一般有三种常见的成员：属性field、方法method和构造器constructor；

1.2.3 实例

1.3两者之间的关系

  • 对象是具体的事物，类是对对象的抽象；
  • 类可以看成一类对象的模板，对象可以看做该类的一个具体实例；

2.面向对象的内存分析

Java虚拟机的内存可以分为三个区域：栈stack、堆heap和方法区method area；

2.1栈的特点

  • 方法执行的内存模型，每个方法被调用在栈中都会创建一个栈帧；
  • JVM为每个线程都创建一个栈，栈属于线程私有，不同线程无法实现栈的共享；
  • 栈的存储特点是“先进后出，后进先出”；
  • 栈是一个连续的内存空间，由系统自动分配；

2.2堆的特点

  • 堆用于存储创建好的对象；
  • JVM只有一个堆，被所有线程共享
  • 堆是一个不连续的内存空间；

2.3方法区（静态区）的特点

  • 方法区实际上属于堆，用于存储程序中永远不变或唯一的内容，如静态变量、静态方法和字符串常量等；
  • JVM只有一个方法区，被所有线程共享；

3.this关键字

3.1 对象创建的过程

创建一个对象分为如下四步：

  1. 分配对象空间，并将对象成员变量进行初始化；
  2. 执行属性值的显示初始化；
  3. 执行构造方法；
  4. 返回对象的地址给相关的变量；

3.2this的本质

this的本质就是“创建好的对象的地址”，由于在构造方法调用前，对象就已经创建，因此可以在构造方法中使用this代表“当前对象”；

3.3 this的用法

3.3.1this代表“当前对象”

在程序中产生歧义时，应使用this来指明；

在普通方法中 ，this总是指向调用该方法的对象；

在构造方法中，this总是指向正要初始化的对象；

4.static关键字

4.1static使用特点

  • 用static声明的成员变量为静态成员变量，也称类变量；
  • 静态成员变量被该类的所有对象共享；
  • 可以直接通过类名来访问静态成员；
  • 在static方法中不可以直接访问非static的成员；

4.2static核心要点

  • static修饰的成员变量和方法，从属于类；
  • 普通变量和方法从属于对象；

4.方法

4.1方法的概念

  • 方法用于定义该类或该类的实例的行为特征和功能实现；
  • 方法是类和对象行为特征的抽象；
  • 方法类似于面向过程中的函数，在面向对象中，整个程序的基本单位是类，方法是从属于类和对象的；

4.2方法声明

修饰符 返回值类型 方法名（形式参数列表）{

修饰符：private、default、protected、public和static等等；

返回值类型：有返回值（int，String，double等等）和无返回值（void）;

方法名：命名规则遵循“驼峰原则”；

形式参数列表：用于接收外接传入的数据；

4.3方法重载

构成方法重载的注意事项：

  1. 不同的含义：形参类型、形参个数和形参顺序不同；
  2. 只有返回值不同，不构成方法的重载；
  3. 只有形参的名称不同，不构成方法的重载；

数据类型和运算符——Java（一）

1.标识符使用规范

1.1规则

  • 标识符必须以字母、下划线_和美元符号$开头；
  • 标识符的其他部分可以是字母、下划线_、美元符号$和数字的任意组合；

1.2使用规范

  1. 类名的标识符：每个单词的首字母大写，如GoodMan;
  2. 方法的标识符：第一个单词的首字母小写，后面单词的首字母大写，遵循驼峰原则，如eatFood;
  3. 变量的标识符：第一个单词的首字母小写，后面单词的首字母大写，遵循驼峰原则，如studentAge;
  4. 常量的标识符：全部大写，单词间用下划线隔开，如MAX_Value;

注：中文也属于字母的一种；关键字（保留字）不可作为标识符

2.变量

1.1基本类型与引用类型

  • 基本类型：分为三类——整数类型、字符类型和布尔类型；
  • 引用类型：又称为对象类型，分为五类——类、接口、数组、枚举和标注；

注：基本数据类型根据数据类型不同，其大小也不一样；引用数据类型大小统一，占4个字节，用于存放引用对象地址（1bit(字节)=8byte(位)）；

1.2变量分类

  • 局部变量：从属于方法/语句块
  • 成员变量（实例变量）：从属于对象
  • 静态变量（类变量）：从属于类

1.3基本数据类型

注：

  1. Java语言的整型常数默认类型为int型，声明long型常量需在其后加‘L’；
  2. 浮点数值默认类型为double型，声明float型常量需在其后加’F’；
  3. 字符型数值用单引号，字符串数值用双引号；（字符型数据占2个字节）；
  4. 布尔型数据占1位；

3.运算符

3.1算数运算符

3.1.1整数运算
  • 如果两个操作数中有一个为long型，则结果也为long型；
  • 如果两个操作数都不为long型，则结果为int型，即使操作数全为short型或byte型；
3.1.2浮点运算
  • 如果两个操作数中有一个为double型，则结果也为double型；
  • 如果两个操作数都为float型，则结果为float型；

3.2赋值运算符

a *= b+3 等价于a = a*(b+3)

3.3逻辑运算符

  • 逻辑与“&”和逻辑或“|”；
  • 短路与“&&”和短路或“||”；

3.4位运算

  • “<<”左移运算符，左移一位相当于乘2：3<<2等价于3*2*2；
  • “>>”右移运算符，右移一位相当于除2：12>>2等价于12/2/2;

3.5字符串连接符

“+”运算符两侧的操作中只要有一个是字符串，系统会将另一个操作数转换为字符串，然后进行连接；

3.6条件运算符

形如：x ? y : z
